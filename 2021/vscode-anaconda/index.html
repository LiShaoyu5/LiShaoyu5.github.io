VSCode中默认打开的终端是系统的cmd，要切换Anconda中的各种环境比较麻烦，因此记录一下将VSCode的默认终端更改为Anaconda Prompt的方法。

在文件-首选项-设置中搜索shell:Window，选择在settings.json中编辑，打开的文件如下：

{
    "workbench.iconTheme": "vscode-icons",
    "editor.accessibilityPageSize": 14,
    "python.showStartPage": false,
    "terminal.integrated.shell.windows": "C:\\Windows\\System32\\cmd.exe"
}

找到Anaconda Prompt的快捷方式，右键选择属性，找到目标，复制如下：%windir%\System32\cmd.exe "/K" E:\Anaconda3\Scripts\activate.bat E:\Anaconda3。

将settings.json修改为：

{
    "workbench.iconTheme": "vscode-icons",
    "editor.accessibilityPageSize": 14,
    "python.showStartPage": false,
    "terminal.integrated.shell.windows": "C:\\Windows\\System32\\cmd.exe",
    "terminal.integrated.shellArgs.windows": [
        "/K",
        "E:\\Anaconda3\\Scripts\\activate.bat E:\\Anaconda3"
    ]
}

重新启动VSCode，打开的终端就切换到了Anaconda Prompt。
